# 크래프톤 정글 캠퍼스 생활 안내 Reference

## 원천 페이지

- URL: https://kraftonjungle.notion.site/junglecampuslife
- 기준일: 2026-06-11
- 용도: AI FAQ 게시판의 캠퍼스 생활 안내 원천 자료

## 프로젝트에서 쓰는 방식

이 Notion 페이지는 향후 다음 기능의 기준 자료로 사용한다.

1. 캠퍼스 생활 FAQ 게시글 작성
2. 검색 결과의 기준 문서
3. 비슷한 FAQ 추천
4. RAG 기반 AI 댓글 답변의 근거 문서

## 저장 방식

현재는 원문 URL만 reference로 저장한다.

Notion 원문은 시간이 지나며 바뀔 수 있으므로, 실제 RAG에 넣기 전에는 다음 중 하나의 방식으로 별도 스냅샷을 만든다.

1. Notion에서 Markdown 또는 HTML로 export
2. 중요한 항목만 수동으로 정리한 Markdown 문서 생성
3. 접근 가능한 경우 페이지 내용을 추출해 `docs/references/snapshots/` 아래에 날짜별로 저장

## 주의 사항

- 이 파일은 원문 내용을 복사한 스냅샷이 아니다.
- 현재 기준으로는 링크와 사용 원칙만 기록한다.
- AI 답변 기능을 만들 때는 답변 근거가 어떤 날짜의 reference에서 왔는지 추적할 수 있어야 한다.
